# Moviegram

Learning project from my first teaching session in 2020. A collaborative landing page for a movies & series community app.

## Technologies

- HTML5
- Vanilla CSS (CSS variables, modular section files)
- Vanilla JavaScript
- Google Fonts, Font Awesome, Material Icons
- Animate.css

## Strategies

- BEM-style naming (`page-header__logobox`, `seccion4__celular`, etc.)
- Modular CSS per section (`section1.css` through `section5.css`)
- Responsive design (media queries, `clamp` for typography)
- Pure CSS modal (checkbox hack for login) and mobile sidebar
